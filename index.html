Teya portal — nav exploration

Four navigation structures for the desktop portal, each available as a browsable prototype with home, sales, money, financial documents, and settings pages. Use the top bar to switch variants while browsing; use "Show data signals" to see the reasoning behind each placement.

A

App-faithful (expandable sections)

Home, Sales, Money, Teya AI. Sales and Money expand into Overview / sub-items. Business admin in a tray.

 B

Strict 4-tab parity (kill variant)

Strict 4-tab parity with the app. Everything else hides inside a collapsible More. Kept as a baseline for the decision matrix.

 C

4-tab with desktop-adapted page tabs

Flat 4-tab sidebar; Sales and Money expose their sub-views as horizontal page tabs with desktop density (filters, bulk actions, reports).

 D

Back-office grouped sidebar

Operations / Money / Business / Records & insights. Dense flat sidebar aimed at finance admins doing deep-work sessions.
